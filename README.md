# ASP.NET Core Authentication System

This project is a simple and secure authentication system built using "ASP.NET Core MVC".

## Features
- User Login and Logout and Signup.
- Cookie-based Authentication
- "ClaimsPrincipal" for storing logged-in user identity
- Secure session handling
- "Toastr.js" for success and error notifications
- Clean and beginner-friendly structure
